## Mock Shopper App

A basic implementation of a shopper signup workflow, with some json endpoints.

### Setup

```
# Clone the project:
git clone git@github.com:alexneigher/shopper.git

# Install the gems
bundle install

# Precompile the assets
rake assets:precompile

# Create the database
rake db:setup
rake db:seed

# Run the app
rails s
```
